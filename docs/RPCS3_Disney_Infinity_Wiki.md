## Configuration

Options that deviate from [RPCS3's default settings]() and provide the best experience with this title are listed below. 

### GPU configuration

| Setting | Option | Notes |
| --- | --- | --- |
| **Write color buffers** | On | Improves overly bright lighting |

### Advanced configuration

| Setting | Option | Notes |
| --- | --- | --- |
| **Read depth buffers** | On | Improves overly bright lighting |
| **Write depth buffers** | On | Improves overly bright lighting |
| **Read color buffers** | On | Improves overly bright lighting |
| **Handle RSX memory tiling** | On | Reduces blocky graphics |

## Known Issues

### Graphical issues

*   [Issue 14572](): There are some graphical glitches present, even with recommended configuration settings.

## Special Notes

The Disney Infinity series requires the use of a real Infinity Base with figures or the built-in Infinity Manager under Manage > Infinity Base.
